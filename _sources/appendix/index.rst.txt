Appendix
========


The main tutorial path ends with Flash Attention. The pages below are
supporting material: environment setup, optional practice kernels,
debugging notes, and reference material for TIRX syntax and APIs.

Use this section as needed:

- **Environment Setup** if the environment or build fails.
- **Practice Kernels** if you want more examples outside GEMM and Flash
  Attention.
- **LLM-Assisted Debugging** if you want prompting and bug-pattern
  checks for kernel work.
- **TIRX API Quick Reference** when you need a compact lookup for
  functions used in the tutorial.
- **TIRX Language Reference** when you want to understand parsing,
  buffers, scopes, metaprogramming, or the compile pipeline.
